ShowLog

ShowLog is my first complex solo coding project from CMPS 262 Advanced Programming. It's a TV show logging platform that is heavily inspired by Letterboxd, featuring user reviews, watchlists, editing features, and more. It operates using APIs and local/online databases.

This project provides a realistic analysis that managerial accountants perform regularly. This project is a basic example of an accounting cycle, which includes posting journal entries, using T-accounts, and creating a trial balance, financial statements, and common-size financial statements. Additionally, there's an analysis that discusses ratios that reveal the liquidity, profitability, and effectiveness of the company
